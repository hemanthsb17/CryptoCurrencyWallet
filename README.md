# Crypto Wallet

A secure cryptocurrency wallet application built using Node.js, Express.js, MongoDB, and MetaMask.

## Features

- User Registration
- User Login
- MetaMask Wallet Integration
- Send and Receive Cryptocurrency
- Wallet Balance
- Transaction History
- Secure Authentication

## Tech Stack

- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

## Installation

```bash
npm install
```

Run the project

```bash
npm start
```
